## Overview

FireAlpaca is a Windows-oriented digital art program made for sketching, painting, illustration, and related creative work. It is aimed at artists who prefer an easy-to-learn interface without giving up the tools needed for more detailed projects, including concept art, character design, comic pages, and animation practice.

The application focuses on a comfortable drawing flow, solid layer handling, and tablet support. Features like brush smoothing, perspective rulers, and non-destructive filters help reduce repetitive cleanup while keeping precise edits within reach.

## How to Use It

Begin with a new canvas and pick a brush preset that fits your process. FireAlpaca works well for rough sketching, clean line art, layered coloring, and comic layout tasks.

Common workflow examples:

- Use a brush with pressure sensitivity enabled for sketching and ink lines.
- Create separate layers for rough art, line art, flat colors, and shading.
- Apply blending modes when building highlights, shadows, or texture.
- Set a perspective ruler before drawing backgrounds or architectural scenes.
- Use non-destructive filters when you want to keep the original artwork editable.
- Save workspace profiles if you switch between painting, comics, and animation tasks.
